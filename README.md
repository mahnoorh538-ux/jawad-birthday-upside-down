# Jawad's Birthday

This repository contains the assets and styles for the Jawad's Birthday event.

## Assets
- `style.css` - Contains styling for the event layout.
- `script.js` - JavaScript files for interactive elements.
- Assets Folder - Contains various images used for the event, excluding user images.

## Included Assets
- `assets/fog.png`
- `assets/grain.png`
- `assets/favicon.png`
- `assets/sound_on.png`
- `assets/sound_off.png`
- `assets/surprise.jpg`
